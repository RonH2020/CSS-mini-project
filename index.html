Flexbox Row

Use these three properties to create a flexbox row layout. 
                    .row {
                    display: flex;
                    flex-direction: row;
                    flex-wrap: wrap;
                  }

Flexbox Column

Use these two properties to create a flexbox column layout. 
                    .column {
                        display: flex;
                        flex-direction: column
                      }
